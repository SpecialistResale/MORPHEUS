# MORPHEUS

**Hyyryn Trust Protocol** — Marketing & landing website.

Built with vanilla HTML, CSS, and JavaScript. No build tools required — deploy directly to GitHub Pages or any static host.

## Structure

```
morpheus/
├── index.html          # Main page
└── src/
    ├── css/styles.css  # Full design system
    └── js/main.js      # Theme, animations, interactions
```

## Features
- Dark / light mode toggle (persisted in localStorage)
- Fully responsive — mobile, tablet, desktop
- Scroll-triggered fade-in animations
- Counter animations on metrics
- Sticky navbar with blur backdrop

## Deploy to GitHub Pages
Push to `main` branch → Settings → Pages → Source: `main / (root)`
